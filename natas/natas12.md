The page has a link to its redacted source code. Page is a file upload form, form has a hidden `filename` parameter which is generated by the server. The `filename` parameter has the form `<random string>.jpg`.

When processing the file upload, PHP code on the server generates a random path where the file upload will be copied to. The random path has the form `upload/<random string>.<extension>` where \<extension\> is the extension from the `filename` parameter.

In normal use the extension is `.jpg` however we can send a crafted request to the server with a `filename` of our choosing, so we can control the extension of the file upload on the server.
